#Perpustakaan Mini

Project ini adalah program Sistem Peminjaman Buku Perpustakaan Mini yang dibuat dengan konsep modular programming. Program ini membantu pustakawan mengelola data buku, anggota, serta aktivitas peminjaman dengan menyimpan semua data ke file `.txt`.

## Fitur Utama

* Tambah, hapus, dan edit data buku
* Kategorisasi buku**: Fiksi, Non-Fiksi, Komik, Majalah, Referensi
* Peminjaman & pengembalian buku + perhitungan denda otomatis
* Cari buku** berdasarkan judul, penulis, atau kategori
* Pendaftaran anggota perpustakaan
* Riwayat peminjaman anggota
* Status buku** (Tersedia / Dipinjam)
* Ranking buku paling sering dipinjam
* Pengaturan durasi peminjaman maksimal
* Login pustakawan
* Semua data tersimpan ke **file eksternal `.txt`

## Cara Menjalankan

1. Compile semua file `.c` menggunakan compiler C
2. Jalankan file executable
3. Ikuti menu yang sudah disediakan di dalam program

## Data File

Program otomatis membuat/menggunakan beberapa file `.txt` seperti:

* `buku.txt`
* `anggota.txt`
* `peminjaman.txt`
* `riwayat.txt`
  dan lainnya sesuai kebutuhan fitur.
